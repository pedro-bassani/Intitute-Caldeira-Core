Steps to Create a Simple Website with an Image

  1. Create a new HTML file (e.g., index.html).
  2. Add the basic HTML structure with <!DOCTYPE html>, <html>, <head>, and <body> tags.
  3. Set the page title using the <title> tag inside the <head>.
  4. Insert a heading or introductory text using the <h1> to <h6> tags.
  5. Use the <img> tag to add an image:
    • Example: <img src="image.jpg" alt="My Image" width="300">
  6. Save the file and open it in a web browser to see the result.
